Three key features of HTML

  • Semantic Structure:
    HTML provides a set of semantic tags like
    ` < header>`,
    ` < nav >`,
    `< main >`,
    ` < section>`,
    and more, which give meaning and structure to the content of a webpage.
  • Images:
    With the ` <img >` tag, HTML enables the inclusion of images on webpages, enhancing visual content and user experience.
  • Forms:
    HTML provides form elements like
    `< input >`,
    ` <select >`,
    ` <textarea >`
    to create interactive forms, enabling user input and data submission.
